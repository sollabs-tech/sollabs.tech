# Sol Labs brand mark

Direction: laboratory stamp (SL monogram).  
References: FedEx hinge + Stripe/Airbnb accent scarcity.  
Palette refs: boutique GCP partner sites (Zencore / Buoyant) — light surface, navy type, blue CTA.

## Anti-Solana test

No purple/magenta/neon green, no gradient orb, no sun rays, no three-bar S.

## Wordmark

**SOLLABS** — FedEx hinge; accent reserved for CTA only.

| Attribute | Value |
|---|---|
| Tracking | `0.04em` |
| Colors | SOL = navy; LABS = secondary; CTA = `#0B57D0` |

Prose: **Sol Labs**. Lockup: hinged compound.

## Palette

| Token | Value | Role |
|---|---|---|
| Ground | `#F4F6F9` | Page |
| Elevated | `#FFFFFF` | Panels |
| Label | `#0B1F33` | Navy type |
| Secondary | `#3D5266` | Hinge / body |
